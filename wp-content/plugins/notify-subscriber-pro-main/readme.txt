Post notification for subscribed users. 

== Description ==

“Notify Subscribers” plugin gives you an option to send post notification to Subscribers.
Users by various ways can send an email notification containing post fields.


Features
•   Easy installation
•	Post notification for the subscribed users
•	Post notification for the custom and standard post type.
•	Enable and disable notify for the custom and standard post types.
•	Export user details via (CSV & PDF).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

Upload via WordPress Admin Panel

1.Log into WordPress and go to Plugins page
2.Select the .zip file you downloaded and click Install
3.Enter your FTP information if required and install the plugin
4.Activate the plugin

Upload via FTP

1.Unzip the .zip file you downloaded.
2.Upload the “Notify Subscribers” plugin folder to your "wp-content/plugins" directory.
3.Navigate to your Admin Panel: Plugins.
4.Under “Notify Subscribers”, click Activate.


== Frequently Asked Questions ==

= When notification mail will send? =

When you checked the checkbox in the particular post which related to any custom or standard post type and update/publish it, Notify subscriber will be sent an email to the subscriber.

= How to export user data? =

Go to user detail tab of the plugin page and you will find the two buttons over there. If you want to export data as CSV then click on "Export to CSV" otherwise click on "Export to PDF" for PDF data.

= How to enable and disable the post type notification? =
Go to plugin sub-page "Options" and by default, all the notification options are enabled for all the post types after you can enable and disable post type as you want or as your desire post types.
